Ideal para negocios de pequeno ou médio porte, o site One Page é um site onde você inclui todas as informações que visitante precisa de forma objetiva e criativas em uma unica pagina.

wireframe

wireframe é uma representação basica da estrutura basica de uma pagina. O ideal, é que ele seja feito da maneira mais simples possivel, mostrando apenas o essencial, como uma especie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o wireframe desta pagina
